# AI Voice Comic Maker / 全自動AIボイスコミックメーカー

## 📖 About / 概要

**AI Voice Comic Maker** is a fully automated pipeline that transforms static 4-koma manga images into dynamic, fully voiced short videos (9:16 aspect ratio) designed for platforms like TikTok, YouTube Shorts, and Instagram Reels.
This system leverages **Gemini 1.5 Flash Vision OCR** for intelligent manga parsing and **VOICEVOX** for expressive character audio generation.

**AI Voice Comic Maker**は、静止画の4コマ漫画を、TikTokやYouTube Shorts向けの「声付き縦型ショート動画（9:16）」に全自動で変換するシステムです。**Gemini 1.5 Flash**による高度な画像解析（OCR・コマ割り認識・感情推定）と、**VOICEVOX**によるフルボイス合成を統合した自律型パイプラインです。

## 🚀 Features / 主要機能

- **One-Click Pipeline (ワンクリックパイプライン)**: Drop your manga image and JSON, and the system automatically analyzes, generates voiceovers, and renders the video without manual intervention.
- **Smart Panel Extraction (高度なコマ切り出し)**: Specifically optimized for the [Nano Banana 2 and ChatGPT Images 2.0 Powered Super AI 4-koma System](https://github.com/FURUYAN1234/nano-banana-pro) format. The system automatically trims headers/footers and isolates each horizontal panel with pixel-perfect precision.
- **Dynamic Camera Work (ダイナミックカメラワーク)**: Automatically detects the speaking character's position and zooms/pans to their specific side of the panel, mimicking dynamic comic viewing experiences.
- **Emotion-Driven Audio (感情連動オーディオ)**: Gemini analyzes the facial expression of the speaker and dynamically adjusts VOICEVOX reading speeds (e.g., angry/surprised characters speak faster) to match the mood.
- **Character Consistency (配役の一貫性)**: Voice ID mapping tracks character features across panels, ensuring that the same character keeps the same voice throughout the video without duplication.

## 📥 Input Requirements / 入力仕様

This system is heavily optimized for images generated by the following system:
**[Nano Banana 2 and ChatGPT Images 2.0 Powered Super AI 4-koma System](https://github.com/FURUYAN1234/nano-banana-pro)**

### 1. Supported Manga Format (対応画像フォーマット)
- **Aspect Ratio**: Portrait (A4 size or similar, `aspectRatio < 0.9`).
- **Layout**: 4 horizontal panels stacked vertically (1 column × 4 rows).
- **Structure**: Can include a top title margin (~6.5%) and bottom footer margin (~3.5%), which the system will intelligently ignore during panel slicing.

### 2. Dialogue Data (`metadata.json`)
While Gemini Vision OCR attempts to read the text automatically, you can place a `metadata.json` alongside your image in the `input/` directory to override the AI's OCR results.

## 🛠️ Architecture / システム構造

The pipeline consists of three autonomous engines:

1. **Gemini Engine (`server.js`)**: 
   - Receives the uploaded image.
   - Executes OCR to extract text, speaker identity, and bubble positions.
   - Detects emotion and assigns stable VOICEVOX character IDs.
2. **Audio Engine (`generate-voiceover.ts`)**:
   - Sends parsed dialogue and dynamic speed parameters to the local VOICEVOX engine.
   - Generates and downloads `.wav` files into the compilation directory.
3. **Render Engine (Remotion / `VoiceComic.tsx`)**:
   - Slices the source image using `sharp`.
   - Assembles the panels, synchronizes the audio tracks, overlays subtitles, and applies Ken Burns panning/zooming effects.
   - Compiles the final `v1.1.0` branded `.mp4` video.

## 💻 Setup & Installation / 導入手順

### Prerequisites
- **Node.js**: v20 LTS or higher
- **VOICEVOX**: Installed and running locally (default: `http://127.0.0.1:50021`)
- **Gemini API Key**: Required for image analysis

### Installation
\`\`\`bash
# 1. Clone the repository
git clone https://github.com/FURUYAN1234/ai-voice-comic-maker.git
cd ai-voice-comic-maker

# 2. Install dependencies
npm install

# 3. Configure API Key
# Create a .env file and add your key:
GEMINI_API_KEY=your_api_key_here
\`\`\`

## 🎮 Usage / 使い方

1. Ensure **VOICEVOX** is running on your machine.
2. Start the integrated dev server:
   \`\`\`bash
   npm run dev
   \`\`\`
3. Open your browser to `http://localhost:3000`.
4. Upload your 4-koma manga image (A4 vertical format).
5. The pipeline will automatically parse the image, generate voices, and show a live preview of the video.
6. Click **"Render Video"** to export the final `.mp4` file.

## 📝 Changelog / 更新履歴

### v1.1.0 (2026-05-15)

- **Audio Naturalization**: Removed aggressive pitch manipulation from the emotion engine, relying solely on speed pacing for more natural delivery.

- Initial pipeline integration of Remotion and Gemini OCR.
- Basic dialogue synthesis via VOICEVOX API.
